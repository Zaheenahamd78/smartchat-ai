# 🤖 SmartChat AI

An intelligent chatbot powered by **Groq's Llama 3.3** model. Built with **Node.js + Express** on backend and vanilla **HTML/CSS/JS** on frontend. Deployed on **Render**.

---

## 🌐 Live Demo

👉 **https://smartchat-ai-12ij.onrender.com**

> **Note:** Free tier pe host hai, agar inactive ho to pehli request mein 15-30 second lag sakte hain.

---

## ✨ Features

- 💬 Real-time chat interface
- 🧠 Groq Llama 3.3 API
- 📱 Responsive design
- ⚡ Easy to deploy

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|

| Backend | Node.js, Express.js |
